# Ramp-Up Assignment

## Responsive Portfolio Website

### Objective
Build a mobile-responsive portfolio website using React and TypeScript, following the best code and Git practices followed at Miraiedge.


### Local setup on your machine
`Node - v20.10.0`

`NPM - v0.39.3`




## How to run

**Install all packages:** npm install

**Run on dev mode:** npm run dev

**Run on production mode:** npm run build && npm run start
